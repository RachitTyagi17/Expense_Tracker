Expense Tracker

Welcome to the Expense Tracker application! This Python-based app helps you track your expenses efficiently. Add expenses with categories and descriptions, calculate totals, and visualize your spending with a pie chart.

How to Use the Application

  1. Clone the repository: 
    git clone https://github.com/RachitTyagi17/Expense_Tracker.git
  2. Navigate to the project directory: 
    cd Expense_Tracker
  3. Install the required dependencies: 
    pip install matplotlib
  4. Run the application: 
    python expense_tracker.py
  5. The application window will open. Start tracking your expenses!

Features

  • Add expenses with a category, amount, and description.
  • View a list of all expenses.
  • Calculate total expenses.
  • Visualize spending with a pie chart.
